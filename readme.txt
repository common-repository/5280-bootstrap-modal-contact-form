Quickly and easily add a pop-up contact form to your website. 

== Description ==

5280 Bootstrap Modal Contact Form is a lightweight plugin that adds a pop-up contact form to your website. 

Simply paste the following shortcode [sbmcf] into your website where you want the button that triggers the pop-up to appear. 

You can choose to use the default form or customize your own.

Easily change color of button and text to match the look of your site.

Note: Requires the installation of the Bootstrap Framework, which can be found [here](https://getbootstrap.com/).

## Usage

* Paste the following shortcode [sbmcf] where you want the button that triggers the pop-up to appear.

== Installation ==

* Upload the '5280-bootstrap-modal-contact-form' folder to your '/wp-content/plugins/' directory.
* Activate.
